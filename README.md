# TruLoad Logistics App

A simple BlackBuck-like logistics app with:
- Driver + customer login
- Post load flow for customers
- Truck booking for drivers
- Real-time tracking simulation via Socket.IO
- Mobile-friendly React UI

## Tech Stack
- Frontend: React + Vite
- Backend: Node.js + Express + Socket.IO

## Demo Credentials
- Customer: `9000000002 / cust123`
- Driver: `9000000001 / driver123`

## Run locally

### 1) Backend
```bash
cd backend
npm install
npm run start
```
Backend runs on `http://localhost:4000`.

### 2) Frontend
```bash
cd frontend
npm install
npm run dev
```
Frontend runs on `http://localhost:5173`.
